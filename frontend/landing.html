Built with Modern Tech

WalletWise leverages powerful tools for a robust and scalable foundation.

  • Supabase: Open-source backend providing PostgreSQL database, real-time updates, and secure user authentication.
  • Ethereum & Solidity: Smart contract on the Holesky testnet handles simulated ETH deposits securely.
  • MetaMask: Browser extension integration for user-controlled blockchain transaction signing.
  • Vanilla JS, HTML, CSS: Lightweight and efficient frontend for a responsive user experience.

Security is implemented through Supabase Auth (with password hashing) and user-controlled keys via MetaMask for blockchain operations.
